# **7. PEDIATRIC PATHOLOGY REASONING FRAMEWORK**

1. Consider age and developmental stage  
2. Identify organ and maturation status  
3. Assess architecture + cell type  
4. Identify hallmark pediatric tumor patterns  
5. Apply IHC conceptually (NOT diagnostically)  
6. Add molecular context (high-level only)  
7. Integrate patterns → broad diagnostic family  
8. Recognize pediatric-specific pitfalls

- Mistaking immature tissue for malignant  
- Misclassifying SRBCTs  
- Overinterpreting fetal liver hematopoiesis  
- Misidentifying nephrogenic rests
